# Kunaal Shivakumar Portfolio

Modern personal portfolio built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Lucide Icons.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Assets

Add these files in `public/` before publishing:

- `profile.jpg` for the formal hero portrait.
- `Kunaal_Shivakumar_Resume.pdf` for the resume download button.
- Replace certificate gallery placeholders with real certificate images when available.

## Production

```bash
npm run build
npm run start
```

The project is ready to deploy on Vercel. Push this `portfolio_websitefolder` directory to GitHub, import the repository in Vercel, and use the default Next.js settings.
